# Zepto AI Discovery Engine

This repository contains the initial Phase 1 foundation for the Zepto AI Discovery Engine.

## Phase 1 goals
- Set up project structure
- Define core data models
- Create configuration placeholders for ingestion and processing
- Provide a runnable smoke test for the foundation

## Structure
- src/zepto_discovery/: application package
- tests/: regression and smoke tests
- data/: raw and processed data folders
